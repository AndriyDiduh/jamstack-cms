# JAMstack CMS

End to end serverless blogging system.

Built with Gatsby & AWS Amplify.

### Features

- Authentication
- Secure, signed images
- Admin panel
- Comments
- Server-side rendering

### Roadmap

- Comments
- Video support
- Dynamic logo from Admin
- Analytics dashboard
- Post categories
- WYSWYG
